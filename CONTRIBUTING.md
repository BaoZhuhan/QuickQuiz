# 题目贡献指南 (Contributing Guide)

感谢你对 **QuickQuiz** 的关注！无论是专业课、公选课还是自命题题库，你的贡献都能帮助更多同学高效复习。

我们为你准备了两条贡献路径，无论你是“电脑小白”还是“代码大神”，都能轻松参与。

---

## 🛠 第一步：准备题目数据

首先，请准备好你的原始题面（来自 Word、PDF 或网页截图文字）。
**格式要求**：只要肉眼能看清题干、选项和答案即可，无需手动排版。

---

## 🤖 第二步：利用 AI 转换为标准格式

我们将使用 AI（如 ChatGPT, Claude, Gemini 或 通义千问等）将松散的文本转换为系统可识别的 `.json` 文件。

**请复制下方提示词发送给 AI：**

> **提示词开始**
>
> 你是一位专业的数据工程专家。请将我提供的题目文本转换为一个 JSON 数组。
>
> **JSON 结构要求：**
> ```json
> [
>   {
>     "id": 1,
>     "type": "single", // tf(判断), single(单选), multiple(多选)
>     "text": "题目内容",
>     "options": [
>       { "key": "A", "text": "选项内容" },
>       { "key": "B", "text": "选项内容" }
>     ],
>     "answer": ["A"], // 即使是单选也请放入数组
>     "score": 10      // 单选/判断10分，多选20分
>   }
> ]
> ```
>
> **转换逻辑：**
> 1. 类型判定：含有“多选”字样设为 `multiple`；只有对错设为 `tf`；其余为 `single`。
> 2. 清洗文本：去除题目开头的序号（如 1. 2.）和多余的括号。
> 3. 判断题特殊处理：`options` 固定为 `[{"key":"T","text":"正确"},{"key":"F","text":"错误"}]`，答案对应填入 `["T"]` 或 `["F"]`。
> 4. 输出要求：只输出 JSON 代码块，不要文字说明。
>
> **待处理题目如下：**
> [在此粘贴你的题目原文]
>
> **提示词结束**

## 📤 第三步：提交你的贡献

根据你的技术背景，选择以下一种方式：

### 方案 A：我是小白（邮件分享）
如果你不熟悉 GitHub 操作：
1. 将 AI 生成的代码保存为名为 `文件名.json` 的文件。
2. 将该文件作为附件发送至：**hengyuhan3762@gmail.com**，或者添加作者钉钉**19818073762**
3. 邮件主题请注明：【QuickQuiz 贡献】科目名称 + 你的名字。
4. 我收到后会自动将其整合进项目并在更新日志中致谢。

### 方案 B：我是工科生/开发者（GitHub PR）
推荐直接通过 Pull Request 贡献代码：
1. **Fork** 本仓库。
2. 将 JSON 文件放入 `/data` 文件夹内。
3. 修改 `/data/index.json`：
   - 如果是现有科目，将文件名加入对应科目的 `files` 数组。
   - 如果是新科目，在 `subjects` 数组中新增一个科目对象。
4. 提交 **Pull Request**。

## 📝 贡献准则
- **数据准确性**：提交前请核对一遍 AI 是否漏掉了题目或转换错了答案。
- **开源精神**：提交题目即代表你同意将该题库以 GPL-3.0 协议共享给所有同学使用。
- **文件名规范**：建议使用小写字母或拼音命名（如 `marxism_quiz.json`）。

再次感谢你的贡献！祝期末顺利，绩点 5.0！🌱
